# chb
"C Hand Book" ou, "Manual de C" é algo que estou criando para ajudar meus amigos a aprenderem a linguagem de programação C. Será simples: colocarei códigos completos e trechos de código na pasta "codes".
Eles estão subdivididos em partes para facilitar a busca, e ambos estão nomeados e citados no guia.

# Créditos
Este é um registro de conhecimento e aprendizado como um todo, portanto, haverá erros, equívocos e alguns conselhos não tão bons. Por favor, se encontrar algum erro ou algo duvidoso, relate-o.

No momento, este guia é fortemente inspirado e baseado nos seguintes livros:
Effective C, de Robert C. Seacord.
The Linux Programming Interface, de Michael Kerrisk.


Este guia também se baseia em conhecimento prévio, conteúdo de sites e outras fontes. Cada fonte será divulgada nos capítulos em que aparece.
